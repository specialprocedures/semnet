# Installation

## Requirements

- Python 3.8+
- networkx
- annoy
- numpy
- pandas
- tqdm

Optional for examples:
- sentence-transformers

## Install from PyPI

```bash
pip install semnet
```

## Install from Source

```bash
git clone https://github.com/specialprocedures/semnet.git
cd semnet
pip install -e ".[dev]"
```
